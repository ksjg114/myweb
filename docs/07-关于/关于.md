# 关于本指南

![校园风光](https://www.cdu.edu.cn/__local/0/9C/B4/43E19399EE98F734E4B1D4DAE43_86AB9493_4F5BD.jpg)

## 这是什么

这是一个面向成都大学新生的**非官方指南**。目标不是替代官方通知，而是把新生最常遇到的问题整理成更好读、更好查的文档。

## 内容原则

- **简洁**：少写套话，优先写能直接行动的信息。
- **可验证**：政策、地址、入口尽量链接官方来源。
- **不过期**：店铺、活动、价格变化快，尽量写方法而不是死名单。
- **学生视角**：保留真实使用建议，但不传播未经核实的传闻。
