# 报到流程

![校园风光](https://www.cdu.edu.cn/__local/2/F5/09/D370DB5086A8DBE6E461D90739B_5FB9C4C8_1E3FA.jpg)

## 到校当天按这个顺序走

1. **进校找学院迎新点**：看路牌、问志愿者，别自己瞎绕。
2. **核验身份**：身份证、录取通知书、报到信息。
3. **领取材料**：校园卡/宿舍信息/学院资料按现场安排领取。
4. **去宿舍**：先放行李，再慢慢整理。
5. **看群通知**：班会、体检、军训、入学教育通常会接着安排。

## 报到当天建议

| 场景 | 建议 |
| --- | --- |
| 家长同行 | 先约好集合点，避免人多走散 |
| 行李很多 | 优先送宿舍，不要拖着行李办所有事 |
| 找不到路 | 直接问志愿者/保安/学院迎新点 |
| 信息冲突 | 以学院通知和现场老师为准 |

:::warning 别急着交钱
除学校官方缴费渠道外，不要向陌生个人转账。遇到“代办校园卡/床品/网费”的推销，先问辅导员。
:::
