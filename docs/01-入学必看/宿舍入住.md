# 宿舍入住

![校园风光](https://www.cdu.edu.cn/__local/E/20/63/343FE158EB77F0389346D55433B_5D45791E_1D62C.jpg)

## 入住第一小时

- 核对楼栋、房间、床位。
- 检查门锁、灯、桌椅、空调/风扇、插座、卫生间设施。
- 有损坏先拍照，再按宿舍报修渠道处理。
- 贵重物品随身带或锁好。

## 室友相处建议

| 事情 | 建议 |
| --- | --- |
| 作息 | 第一天就沟通熄灯、闹钟、打电话时间 |
| 卫生 | 排一个简单值日表，越早越省事 |
| 外放 | 视频/游戏尽量戴耳机 |
| 公共物品 | 谁买、谁用、怎么分摊先说清 |

## 不建议

- 私拉电线、使用违规大功率电器。
- 把贵重物品随意放桌面。
- 刚认识就借大额钱或共用账号。
