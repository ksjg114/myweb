# 报到清单

![校园风光](https://www.cdu.edu.cn/__local/E/37/C7/5B6C7429DC397939790EED4E4B2_99C92B5C_16DB6.jpg)

## 必带材料

| 类别 | 建议 |
| --- | --- |
| 身份证明 | 身份证原件；复印件可多备几份 |
| 录取材料 | 录取通知书、报到相关通知截图/纸质版 |
| 档案党团 | 档案、团员证/团组织关系等按学院通知办理 |
| 照片 | 一寸/两寸证件照；电子版也保存一份 |
| 银行/缴费 | 银行卡、手机支付、缴费凭证截图 |

## 生活用品：别一次性搬家

- **先带**：洗漱用品、换洗衣物、拖鞋、充电器、插排（看宿舍规定）、常用药。
- **到校买**：脸盆、衣架、收纳盒、清洁工具、床上用品。
- **不建议带太多**：大件行李、锅具、大功率电器。

## 到校前 24 小时

- 再看一遍学院群/辅导员通知。
- 把证件和材料放进一个文件袋。
- 导航目的地建议填：**成都大学 / 成都市成洛大道2025号**。

:::tip 快速原则
证件和通知最重要；生活用品可以到校补，材料缺了最麻烦。
:::
