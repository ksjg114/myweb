# 新生常见坑

![校园风光](https://www.cdu.edu.cn/__local/3/4B/23/DCE9354273E5831CFB44E1FD76E_402E8DB5_1562A.jpg)

## 最容易踩的坑

| 坑 | 怎么避 |
| --- | --- |
| 低价校园卡/流量卡推销 | 不急着办，先问辅导员和学长学姐真实体验 |
| “必须买”的床品/资料 | 看学校/学院是否明确要求，不确定就不付款 |
| 二手群捡漏 | 面交验货，别先款；电子产品尤其谨慎 |
| 陌生人代办缴费 | 只走官方缴费渠道 |
| 群里假通知 | 认准辅导员、学院官方群、学校官网 |

## 安全提醒

- 身份证、学生证、银行卡不要随便拍给别人。
- 寝室门随手关，晚上回宿舍结伴更稳。
- 遇到纠纷别硬刚，先找辅导员/宿管/保卫处。

:::tip 判断信息靠三问
谁发的？要不要钱？能不能在官网/学院通知里找到？
:::
