# 关于本站

这是一个面向成都大学新生的**非官方新生指南网站**。它不是学校正式通知平台，而是把新生常问的问题整理成更容易阅读、检索和更新的文档。

## 我们想解决什么问题

- **报到前少迷路**：把地铁站、火车站 / 机场到校路线、校园地图入口放在一起。
- **到校后少踩坑**：把食堂支付、请假、校园跑、军训、宿舍门禁等容易漏看的提醒单独整理。
- **后续方便补内容**：宿舍、课程、食堂、公共设施、FAQ 都按大类拆开，后续可以逐篇更新。

## 信息来源和准确性

本站内容主要来自：成都大学官网、成都大学新闻网、成都大学校园服务入口、学生经验整理以及用户提供的大纲。涉及报到、缴费、住宿、军训、请假、课程安排等事项时，**最终以成都大学官网、学院通知、辅导员通知、任课教师要求和现场公告为准**。

如果你发现页面内容过期、路线不准、食堂档口调整、链接失效或图片无法显示，欢迎反馈。

## 如何反馈或补充

如果还有需要了解的内容，或者你想补充自己的经验，可以到 GitHub 提 Issue：

- [提交问题 / 建议：GitHub Issues](https://github.com/cdu-students/myweb/issues)
- 反馈时建议写清楚：页面名称、你看到的问题、建议改成什么、信息来源或截图。
- 不建议在 Issue 里公开手机号、身份证号、学号、宿舍号等隐私信息。

## 贡献建议

- 写给新生看的内容尽量短句、分点、少废话。
- 不确定的信息要标注“以现场 / 通知为准”。
- 图片优先使用成都大学官网、成都大学新闻网等与学校相关且可公开访问的外链；不要把本地图片、PDF、压缩包等二进制文件直接提交到仓库。

本站为学生向非官方项目，不代表成都大学官方立场。本站只做信息整理和访问入口提示，不承担因信息时效变化、个人理解差异或第三方平台内容变化导致的后果。
