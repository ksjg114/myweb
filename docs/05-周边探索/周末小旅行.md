# 周末小旅行

![校园风光](https://www.cdu.edu.cn/__local/7/52/FA/485DE80A52F585DB64EBA4D33FF_74AE8975_25807.jpg)

## 周末目的地

| 地点 | 适合 | 提醒 |
| --- | --- | --- |
| 青龙湖湿地公园 | 骑行、散步、拍照 | 离学校近，适合半天 |
| 熊猫基地 | 看熊猫、带朋友玩 | 提前预约，早去体验更好 |
| 都江堰 | 一日游、历史景区 | 提前买票/查车次 |
| 青城山 | 爬山、自然风景 | 穿舒服的鞋，带水 |
| 洛带古镇 | 古镇、小吃 | 周末人多 |

## 出行原则

- 第一次周末游别排太满，最多 1-2 个点。
- 先查天气、门票、返程车。
- 带学生证/身份证，很多地方会用到。

:::tip
最适合新生的第一站：青龙湖。近、成本低、失败率低。
:::
