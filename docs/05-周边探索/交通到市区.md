# 交通到市区

![校园风光](https://www.cdu.edu.cn/__local/D/B5/C1/C0D20432D3B72B73CD4BABE2640_36D26FEF_1D09D.jpg)

## 最常用：地铁 4 号线

成都大学附近有地铁 4 号线相关站点，去市区很方便。第一次出门建议直接用地图软件规划路线。

| 目的地 | 大致思路 |
| --- | --- |
| 春熙路/太古里 | 地铁换乘到市中心商圈 |
| 宽窄巷子 | 4 号线方向上可达核心景区附近 |
| 成都东站 | 地铁/打车都可，赶车建议预留更多时间 |
| 熊猫基地 | 地铁换乘 + 公交/打车/景区交通 |

## 出行建议

- 周末进城尽量避开饭点和晚高峰。
- 末班车时间会变，出发前看地图或地铁官方信息。
- 多人同行短距离可打车平摊，夜间更省心。
