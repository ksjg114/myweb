# 成都经典地标

![校园风光](https://www.cdu.edu.cn/__local/7/94/10/EECE0DE9D1717043EC1679CB27B_32C550A7_1ACD6.jpg)

## 新生第一次进城推荐

| 地点 | 适合做什么 | 建议 |
| --- | --- | --- |
| 春熙路/太古里 | 逛街、拍照、吃饭 | 人多，保管好随身物品 |
| 宽窄巷子 | 城市地标、游客路线 | 适合第一次打卡 |
| 人民公园 | 喝茶、体验慢生活 | 周末人多，早点去 |
| 武侯祠/锦里 | 历史文化、夜景 | 可和市区吃饭一起安排 |
| 熊猫基地 | 看大熊猫 | 需要提前看预约和开放时间 |

## 半日路线

- **轻松版**：宽窄巷子 → 人民公园 → 春熙路。
- **拍照版**：太古里 → 镋钯街/玉林附近 → 夜景。
- **游客版**：熊猫基地 → 市区火锅。
