# 宿舍生活

> 本页把宿舍篇内容整理到网站正文中。宿舍门禁、缴费、用电、网络、设备开放等信息会随楼栋和年份调整，最终以宿管、学院、后勤通知和现场公告为准。

## 基础小程序

- **电费缴费**：支付宝 → 完美校园小程序绑定一卡通 → 缴费 → 宿舍预缴电费，选定楼栋、寝室号充值。
- **热水洗浴**：支付宝趣智校园小程序充值、开关热水。
- **洗衣服务**：支付宝胖乖生活小程序使用洗衣机。

## 门禁和开关门时间

- **开门**：早上 6:00。
- **关门**：晚间 23:00，急事可联系宿管阿姨开门。
- **外宿**：需提前向班主任请假，前往宿管处登记；23 点后回寝系统记晚归。
- **进门**：宿舍进门刷一卡通，忘带卡可找阿姨借用备用钥匙。

## 违规电器

- 禁止使用吹风机、烫发棒、电磁炉等大功率电器。

:::warning 提醒
不同楼栋对电器、吹风机、公共空间设备的管理可能不完全一致。不要只听同学口头转述，优先看宿管通知和学院要求。
:::

## 不断电

- 宿舍全天 24 小时供电，额定限电 800W，超载自动断电。
- **首次断电**：5 分钟后自动恢复。
- **二次断电**：永久断电，需去事务大厅核查原因、学院签字盖章后方可复电，同步扣分登记。

## 网络

- 宿舍预留网线接口，可连接校园网；可办理校园卡，支持宽带安装。
- 成都大学校园网连接，可以先看学校“校园服务”页的信息网络中心入口。
- 用宿舍宽带时，优先按运营商 / 校园卡办理点给出的账号、拨号或认证方式操作。
- 如果没有网线，可以试试在教学区域（CDU 信号很充足的地方）用电脑连接。
- 可以老师上完课去用讲台的电脑激活，或者把讲台电脑的网线插你电脑上去连接名为 CDU 的 WiFi 信号，输入一次用户名 + 密码即可。

## 公用设备

- 楼栋设多功能房：配备吹风机、微波炉、平价打印机、洗衣机、烘干机、售货机、熨烫机。
- 部分宿舍标配吹风机，且部分宿舍配有电梯。

## 住宿规格与收费

- **住宿规格**：四人间，床铺尺寸 2.0m × 0.9m，床高 1.2m，分标准上床下桌、六改四房型。
- **收费**：标准间 1200 元 / 年，六改四房型 1000 元 / 年。
