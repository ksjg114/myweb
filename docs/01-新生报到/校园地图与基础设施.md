# 校园地图与基础设施

成都大学官网校园地图 PDF 设置了同源嵌入限制，直接放 `iframe` 可能无法稳定显示。这里改为保留**官方地图入口 + 文字导览**：新生手机端直接点开官方 PDF，配合下面的基础设施表看会更稳。

:::tip 校园地图入口
[打开成都大学官网校园地图 PDF](https://www.cdu.edu.cn/xyfw/xiaoyuanditu.pdf)

手机端建议点开后用浏览器“分享 / 保存到文件”，报到当天离线也能看。
:::

- 如果官网 PDF 打不开，先从 [成都大学校园服务](https://www.cdu.edu.cn/xyfw.htm) 进入“校园地图”。
- 如果地图和现场路线不一致，以迎新现场指引、学校最新通知和道路围挡为准。

## 新生先认这些基础设施

| 设施 | 新生用来干什么 | 到校后怎么找 |
| --- | --- | --- |
| **校门/迎新点** | 报到、问路、确认学院流程 | 报到当天先找学院迎新牌和志愿者 |
| **宿舍区** | 放行李、领取/激活门禁、办理入住 | 以宿舍分配信息为准，先问楼栋宿管 |
| **教学楼/学院楼** | 上课、班会、入学教育 | 第一周提前踩点，别等上课前才找 |
| **图书馆** | 自习、借书、查资料 | 记住开放时间和入馆方式 |
| **食堂** | 三餐、充值/饭卡相关咨询 | 新生先试离宿舍最近的食堂 |
| **快递点** | 取行李、网购物品 | 保存取件码；大件到校后再买更省力 |
| **校医院/医务点** | 小病、体检、军训身体不适 | 需要病假证明时优先走正规医疗渠道 |
| **后勤维修/宿管** | 宿舍报修、门禁、热水、洗衣 | 宿舍楼公告栏和宿管处通常有流程 |
| **教务/学生事务入口** | 选课、证明、请假、学籍事务 | 先查学校“校园服务”页和网上办事大厅 |

## 读地图的小技巧

- **先找自己宿舍楼**，再找最近食堂、快递点、教学楼。
- **先看主路**，不要第一天就抄小路；拖箱、下雨天尤其如此。
- **把地铁站、宿舍、食堂、教学楼做成手机收藏点**，第一周会省很多时间。
- **地图和现实可能有差异**：施工、临时封路、迎新路线以现场为准。
