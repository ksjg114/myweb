# 地铁站：成都大学站 / 明蜀王陵站怎么选

成都大学周边最常用的是 **成都地铁 4 号线**。新生先记住两个站：**成都大学站** 和 **明蜀王陵站**。

:::warning 先说明
不同学院、专业、年级的宿舍可能调整；下面按公开校园地图、地铁站位置和学生常用路线整理。到校当天以学院迎新点、宿舍分配和现场指引为准。
:::

## 两个地铁站怎么选

| 站点 | 更适合去哪 | 适合什么情况 |
| --- | --- | --- |
| **成都大学站** | 学校主出入口、综合性办事点、校内公共区域 | 第一次到校、拖大行李、找迎新点、去教学/行政公共区域 |
| **明蜀王陵站** | 学校东侧区域、靠东门一带 | 已知道自己住东侧宿舍，或要去东侧教学/学院区域 |
| **十陵站** | 校外十陵生活区 | 吃饭、买东西、周边生活，不建议新生第一次拖箱报到走这里 |

## 和宿舍、教学楼的大致关系

| 你的位置/目的 | 推荐站点 | 说明 |
| --- | --- | --- |
| 第一次报到、还不知道宿舍楼 | **成都大学站** | 最稳，先到校门/迎新点确认路线。 |
| 住在东侧宿舍区，如 15—20 栋一带（需以实际分配为准） | **明蜀王陵站** | 公开校园经验中常见说法是东门一带离该站更顺。 |
| 去综合教学区、图书馆、行政/公共服务区域 | **成都大学站** | 更适合先进入校园主区域，再按校园地图走。 |
| 去机械工程学院、东侧学院/楼栋 | **明蜀王陵站** | 东门方向通常更近，但仍建议对照官方校园地图。 |
| 去十陵商圈吃饭/买日用品 | **十陵站 / 成都大学站后步行或骑行** | 看具体店铺位置，不建议凭感觉走。 |

## 新生到站后怎么走

1. **先打开地图软件**：目的地不要只填“成都大学”，尽量填“成都大学 + 宿舍楼/学院/教学楼”。
2. **拖箱优先走大路**：少抄小路，避开台阶、施工围挡和车流。
3. **进校后找志愿者**：报到季一般有迎新指引，直接问“我这个学院/宿舍往哪走”。
4. **不要急着办卡/买东西**：先完成报到和入住，再处理生活用品。

## 地铁站相关来源

- 成都大学官网校园服务页提供“校园地图”入口与学校地址信息。
- 成都大学官网学校简介页显示学校地址为 **成都市成洛大道2025号**。
- 成都大学站、明蜀王陵站均为地铁 4 号线站点；具体运营时间以成都地铁/地图软件为准。
