# 请假与校园跑、军训

:::warning 重要
请假规则每个学院、课程、军训连队可能不同。本页给的是“新生可直接照着做”的通用模板；最终以辅导员、任课老师、体育老师、军训教官通知为准。
:::

## 正课请假

### 什么时候要请假

- 生病、就医、突发家庭事项。
- 学校/学院安排与课程冲突。
- 其他确需离校或缺课的情况。

### 常见申请方式

1. **先联系辅导员**：说明原因、时间、是否离校。
2. **再联系任课老师**：尤其是实验课、考勤严格课程。
3. **按学院要求提交线上/纸质申请**：可能通过网上办事大厅、学院表单、班级请假系统或纸质假条。
4. **保留证明**：病假保留病历、挂号/缴费记录；事假保留必要说明材料。
5. **返校/返课后销假**：不要请完就结束，按要求销假。

### 正课请假消息模板

```text
老师您好，我是XX学院XX专业XX班的XXX，学号XXXX。
因【具体原因：如发热就医/家庭突发事项/学校活动冲突】，需要请假【日期+第几节/课程名称】。
我已/将提交【学院要求的请假申请或证明材料】，并会主动补上课程内容。
请老师批准，谢谢老师。
```

## 军训请假

### 先记住

- 军训通常考勤严格，不要口头说一声就走。
- 身体不适先报告教官/辅导员，不要硬撑。
- 病假尽量有校医院或正规医院证明。

### 军训请假格式

```text
军训请假申请

本人XXX，XX学院XX专业XX班，学号XXXX，所在连/排为XXXX。
因【具体原因】，需于【日期 时间】至【日期 时间】请假。
请假期间去向：【校医院/宿舍休息/医院就诊/其他】。
联系方式：XXXXXXXXXXX。
证明材料：【病历/诊断证明/其他，无则写暂无】。

本人承诺请假期间注意安全，按时归队并销假。
申请人：XXX
日期：XXXX年XX月XX日
```

## 校园跑请假/异常处理

### 常见情况

| 情况 | 建议 |
| --- | --- |
| 生病不能跑 | 先问体育老师/辅导员是否需要病假证明。 |
| 定位异常/记录失败 | 保留截图，尽快联系体育老师或平台客服。 |
| 忘记打卡 | 通常很难补，尽早问课程老师是否有补救方式。 |
| 长期不适合跑步 | 准备医院证明，咨询体育课免修/缓修/替代要求。 |

### 校园跑请假模板

```text
老师您好，我是XX学院XX专业XX班XXX，学号XXXX。
因【身体原因/就医/其他原因】，预计【日期范围】无法完成校园跑。
我已准备【病历/诊断证明/截图等材料】，想咨询是否可以请假、补跑或申请替代安排。
谢谢老师。
```

## 通用提醒

- 不要先斩后奏：能提前请就提前请。
- 不要伪造病假条/截图：后果比缺一次勤严重得多。
- 请假信息写清楚：姓名、学院、班级、学号、时间、原因、去向、联系方式。
- 请假后主动补课/补训练，不要等老师追。
- 涉及离校、夜不归宿、外出就医，一定同步辅导员和室友。
