# 报到当天：按这个顺序做

## 一句话流程

**到校 → 找学院迎新点 → 核验身份 → 拿材料 → 去宿舍 → 找宿管/门禁 → 整理床位 → 看班级群通知。**

## 详细流程

| 步骤 | 做什么 | 提醒 |
| --- | --- | --- |
| 1 | 到校门/迎新点 | 不知道去哪就问志愿者，别拖着箱子乱走。 |
| 2 | 找学院 | 看学院名称，不只看专业名称。 |
| 3 | 身份核验 | 身份证、录取通知书、报到二维码/截图。 |
| 4 | 领取材料 | 可能包括宿舍信息、校园卡/饭卡说明、学院资料。 |
| 5 | 去宿舍 | 先放行李，再办其他事情。 |
| 6 | 领取门卡/门禁 | 常见在宿舍楼下或指定点办理，问宿管最有效。 |
| 7 | 加群/看通知 | 班会、军训、体检、入学教育通常会接着来。 |

## 别当天硬做的事

- 不急着买齐所有生活用品。
- 不急着办陌生推销的电话卡/网卡。
- 不急着评价食堂，先把最近的几个窗口试一遍。
- 不急着参加所有社团群，先把报到和宿舍安顿好。

## 报到当天应急话术

> 老师/同学你好，我是 XX 学院 XX 专业新生，我现在在 XX 位置，想确认报到点/宿舍楼怎么走。

> 老师/宿管您好，我是 XX 宿舍新生，想问门禁/门卡在哪里领取或激活？

> 辅导员您好，我的 XX 材料暂时缺失，是否可以先报到后补交？需要我什么时候补齐？
