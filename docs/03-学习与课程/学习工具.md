## 官方工具

| 工具 | 用途 |
| --- | --- |
| [教务系统](https://jw.cdu.edu.cn/) | 课表、成绩、选课 |
| [图书馆](https://lib.cdu.edu.cn/) | 借书、数据库、学习空间 |
| [校园邮箱](https://mail.cdu.edu.cn/) | 接收学校/课程邮件 |
| [网上办事大厅](https://ehall.cdu.edu.cn/) | 证明、申请、流程办理 |

## 自用工具推荐

- 课程表 App / 日历：避免错过课和考试。
- 云文档：小组作业别靠微信聊天记录。
- Zotero / 文献管理：写论文前早点学。
- GitHub / Gitee：代码、项目、简历作品沉淀。
