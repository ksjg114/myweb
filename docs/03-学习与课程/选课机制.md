# 选课机制

![校园风光](https://www.cdu.edu.cn/__local/1/34/E9/C2296705E405E82DFEB37B43192_32CFCD20_1B611.jpg)

## 选课前先看

| 检查项 | 为什么重要 |
| --- | --- |
| 培养方案 | 确认必修/限选/任选要求 |
| 课程时间 | 避免上课冲突 |
| 课程容量 | 热门课可能很快满 |
| 考核方式 | 闭卷、论文、展示、平时分占比不同 |
| 任课教师通知 | 有些课会提前建群或说明要求 |

## 选课步骤

1. 登录 [教务系统](https://jw.cdu.edu.cn/)。
2. 先确认必修课，再看选修课。
3. 保存/截图选课结果。
4. 开学第一周留意退补选通知。

:::warning
选课时间一过会很被动。把选课开始时间提前设闹钟。
:::
