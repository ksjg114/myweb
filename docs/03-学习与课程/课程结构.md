# 课程结构

![校园风光](https://www.cdu.edu.cn/__local/F/D7/66/76B3B2C403F72136440E690CFB9_2C67BE2F_1379F.jpg)

## 你会遇到的课

| 类型 | 说明 | 建议 |
| --- | --- | --- |
| 公共基础课 | 大学英语、体育、思政等 | 别挂科，绩点从这些课开始累积 |
| 专业基础课 | 专业入门与核心基础 | 认真学，后续课程会反复用到 |
| 专业核心课 | 影响能力和方向 | 重点投入时间 |
| 通识/选修 | 兴趣拓展、学分补充 | 结合兴趣和时间选择 |
| 实践课程 | 实验、实训、课程设计 | 提前组队/准备材料 |

## 大一最重要的事

- 看懂本专业培养方案：毕业要多少学分、哪些课必须过。
- 记住考试周、补考、重修、选课时间。
- 别让“水课”拖绩点，也别只盯绩点忘了能力。
