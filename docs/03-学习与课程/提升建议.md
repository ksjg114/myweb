# 提升建议

![校园风光](https://www.cdu.edu.cn/__local/C/5B/1D/450C5B7D04E8B29C2F339F04098_09F7CBBB_1D328.jpg)

## 大一就能做的事

| 目标 | 做法 |
| --- | --- |
| 稳住绩点 | 平时作业别拖；考试周前两周开始复习 |
| 找方向 | 听学院讲座、问学长学姐、看专业培养方案 |
| 做作品 | 课程项目、比赛、社团项目都可以沉淀成作品 |
| 学英语 | 每周固定输入输出，别只靠考前背单词 |
| 准备竞赛 | 先参加校级/院级，再冲省级/国家级 |

## 不建议

- 盲目报一堆证书。
- 只参加活动不沉淀成果。
- 一开始就把绩点、比赛、社团、兼职全拉满。

:::tip
大学不是只拼“忙”，而是拼你能不能长期稳定地产出。
:::
