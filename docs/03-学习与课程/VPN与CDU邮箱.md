# VPN 与 CDU 邮箱怎么用

这两个入口是新生后面查资料、收通知、访问校内系统时最常用的工具。下面只写**最少够用版**。

## 1. 学校 VPN

**用途**：不在校园网时，访问部分校内资源或图书馆数据库。比如在宿舍外、家里、实习地需要查学校购买的数据库资源时，优先尝试 VPN。

**入口**：[成都大学 VPN 入口](https://vpn.cdu.edu.cn/portal/#!/login)

**使用步骤**：

1. 打开 VPN 入口。
2. 按页面提示使用学校统一身份认证账号登录。
3. 登录后从 VPN 页面内点击要访问的校内系统/资源，不要直接复制内网地址到普通浏览器窗口。
4. 用完后退出登录，尤其是在公共电脑上。

:::tip 常见问题
- 登录失败：先确认学号/统一身份认证密码是否正确；新生账号可能需要等学校统一开通。
- 数据库打不开：先从图书馆或 VPN 门户内进入，再尝试访问数据库。
- 手机端不好用：优先用电脑浏览器处理论文、数据库、教务等复杂操作。
:::

## 2. CDU 校园邮箱

**用途**：接收学校通知、注册部分学习/竞赛/科研平台、对外联系老师或学院。比私人邮箱更正式。

**入口**：[成都大学校园邮箱](https://mail.cdu.edu.cn/)

**使用建议**：

- 第一次登录后，先检查是否需要绑定手机、修改初始密码或完善安全信息。
- 给老师发邮件时，标题写清楚“学院 + 年级 + 姓名 + 事项”，正文少废话。
- 重要附件建议用 PDF，并在正文里说明“附件为……”。
- 不要用校园邮箱注册乱七八糟的网站，避免垃圾邮件过多。

## 3. 推荐收藏的官方入口

| 入口 | 用途 |
| --- | --- |
| [统一信息门户](https://myapp.cdu.edu.cn/) | 汇总校内系统入口 |
| [网上办事大厅](https://ehall.cdu.edu.cn/) | 线上办事、证明、流程申请等 |
| [校园邮箱](https://mail.cdu.edu.cn/) | 学校邮箱登录 |
| [教务系统](https://jw.cdu.edu.cn/) | 课表、成绩、选课等 |
| [信息网络中心](https://nic.cdu.edu.cn/) | 网络、账号、信息化服务相关通知 |

## 信息来源

- 成都大学官网首页与 [校园服务](https://www.cdu.edu.cn/xyfw.htm) 页均列有 VPN 入口、校园邮箱、统一信息门户、网上办事大厅、教务系统和信息网络中心等官方入口；校园服务页中的 VPN 入口指向 `vpn.cdu.edu.cn/portal/#!/login`。
- 具体账号开通时间、初始密码规则、认证方式可能随年级和学校通知调整，以信息网络中心、学院辅导员和迎新材料为准。
