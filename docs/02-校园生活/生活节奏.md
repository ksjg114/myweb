# 生活节奏

![校园风光](https://www.cdu.edu.cn/__local/6/92/92/154DDE9DD2BE4501BAEC3302D54_A7AC8175_3A81A.jpg)

## 大一常见节奏

| 时间段 | 常见安排 |
| --- | --- |
| 早上 | 上课、早八、早饭抢时间 |
| 下午 | 上课、实验、体育、社团招新 |
| 晚上 | 自习、班会、社团、运动 |
| 周末 | 补觉、洗衣、青龙湖/市区走走 |

## 适应建议

- 第一周先把课表、教学楼、通勤时间摸清。
- 给自己留固定自习时间，别全靠考前冲刺。
- 社团先多看少报，确定喜欢再投入时间。
- 每周整理一次宿舍和待办，生活会轻松很多。
