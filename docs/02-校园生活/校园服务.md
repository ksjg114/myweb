# 校园服务

![校园风光](https://www.cdu.edu.cn/__local/7/6F/09/BBB30F46731083AB56CC972DC0B_3EE82FF4_2810C.jpg)

## 常用线上入口

| 入口 | 用途 |
| --- | --- |
| [统一信息门户](https://myapp.cdu.edu.cn/) | 常用校内系统入口 |
| [网上办事大厅](https://ehall.cdu.edu.cn/) | 线上事务办理 |
| [教务系统](https://jw.cdu.edu.cn/) | 课表、选课、成绩 |
| [校园邮箱](https://mail.cdu.edu.cn/) | 学校邮件服务 |
| [图书馆](https://lib.cdu.edu.cn/) | 借阅、数据库、自习资源 |
| [学生事务服务](https://xsswzx.cdu.edu.cn/) | 学生事务相关信息 |

## 办事建议

- 账号密码第一时间保存到密码管理器/备忘录。
- 系统打不开时，先换浏览器、校园网或 VPN/校内入口。
- 涉及证明、盖章、补办，优先查网上办事大厅。

:::tip
能线上办的先线上办；不能线上办，再问学院办公室或辅导员去哪里排队。
:::
