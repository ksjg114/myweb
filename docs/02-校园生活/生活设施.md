# 生活设施

![校园风光](https://www.cdu.edu.cn/__local/1/58/45/3D026FAE8979FF3F638096D958C_5C88C777_1E82B.jpg)

## 高频设施清单

| 设施 | 用途 | 小建议 |
| --- | --- | --- |
| 食堂 | 三餐 | 错峰吃饭，先试热门窗口 |
| 超市/便利店 | 水、零食、日用品 | 大件可网购，别一次买太多 |
| 快递点 | 收发包裹 | 取件码截图保存 |
| 洗衣房 | 洗衣烘干 | 高峰期提前排队/预约 |
| 医务/医院 | 小病处理 | 急症优先去正规医院 |

## 到校后 3 天内建议完成

- 找到离宿舍最近的食堂、超市、快递点。
- 了解宿舍楼洗衣、热水、门禁、报修方式。
- 保存辅导员、宿管、班委联系方式。
