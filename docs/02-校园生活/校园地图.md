# 校园地图

![校园风光](https://www.cdu.edu.cn/__local/1/22/6F/882E3603BBEA404C2E0C4923330_1870F203_1EB33.jpg)

## 先记住 5 类地点

| 类型 | 你会用来做什么 |
| --- | --- |
| 教学楼 | 上课、考试、找老师 |
| 图书馆 | 自习、借书、查资料 |
| 食堂/商业点 | 吃饭、买日用品 |
| 宿舍区 | 日常生活、取快递、洗衣 |
| 体育场馆 | 跑步、球类、社团活动 |

## 新手找路方法

- 第一次上课提前 **20 分钟** 出门。
- 地图软件搜“成都大学 + 建筑/校门/地铁站”。
- 校园里迷路很正常，直接问路效率最高。

## 校外定位

- 学校地址：**成都市成洛大道2025号**。
- 常用公共交通关键词：**成都大学站、十陵、成洛大道**。
