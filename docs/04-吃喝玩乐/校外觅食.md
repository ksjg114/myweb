# 校外觅食

![校园风光](https://www.cdu.edu.cn/__local/0/26/39/DF28CCC454AC7E4A39E58D0F089_0AA14123_1747E.jpg)

## 搜索关键词

- **十陵**：日常聚餐、奶茶、小吃。
- **成都大学站**：地铁口周边，适合顺路吃。
- **青龙湖周边**：散步后吃饭，可和周边探索一起安排。
- **春熙路/太古里**：市区逛街吃饭，预算更高。

## 选店三步

1. 地图软件看距离和营业时间。
2. 点评平台看近期评价，不只看总分。
3. 人均、排队、交通成本一起算。

:::tip
新生聚餐别选太远。刚开学大家时间不稳定，近一点更容易成行。
:::
