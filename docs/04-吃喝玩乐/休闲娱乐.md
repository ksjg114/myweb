# 休闲娱乐

![校园风光](https://www.cdu.edu.cn/__local/9/B7/CA/0ED375CEEAEF766D671411F1361_5C369A2D_22C8D.jpg)

## 校内

| 玩法 | 适合 |
| --- | --- |
| 操场散步/跑步 | 晚上放松、低成本运动 |
| 球类运动 | 约同学、加入社群 |
| 社团活动 | 找兴趣圈子 |
| 图书馆/博物馆/展览 | 安静型周末 |

## 校外

- **青龙湖湿地公园**：近、适合骑行散步。
- **电影院/桌游/密室**：十陵或市区商圈看预算选择。
- **春熙路/太古里**：逛街、拍照、吃饭。

## 省钱建议

- 优先看学生票、团购、工作日优惠。
- 出门前确认末班车，不然打车成本会变高。
