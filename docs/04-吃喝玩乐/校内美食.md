# 校内美食

![校园风光](https://www.cdu.edu.cn/__local/0/FF/96/B4A5637DE1FE3C18CE84A38F6F3_AD625815_10A08.jpg)

## 吃饭策略

| 场景 | 建议 |
| --- | --- |
| 早八 | 提前买面包/牛奶，别卡点排队 |
| 午饭高峰 | 错峰 15 分钟体验会好很多 |
| 不知道吃啥 | 跟着排队长的窗口试一次 |
| 想省钱 | 食堂基础套餐通常最稳 |
| 想换口味 | 校内饮品/小吃 + 校外十陵商圈轮换 |

## 新生别急着评价

食堂窗口会换，口味也因人而异。建议先用一周把常去窗口摸一遍，再固定自己的“安全菜单”。
